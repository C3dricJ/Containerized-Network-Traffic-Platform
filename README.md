# Containerized Network Traffic Platform

A containerized network monitoring platform built with:

- Docker
- Node.js
- PostgreSQL
- Grafana
- Python
- Scapy
- Prometheus

## Features

- Real-time packet capture
- PostgreSQL packet storage
- Grafana dashboards
- Prometheus metrics endpoint
- Dockerized deployment

## Architecture

Packet Capture (Scapy)
        ↓
Node.js API
        ↓
PostgreSQL
        ↓
Grafana Dashboard
